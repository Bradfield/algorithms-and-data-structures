Review Questions
================

Determine the Big-O performance of the following code fragments

```python
for i in range(n):
   for j in range(n):
      k = 2 + 2
```
```python
for i in range(n):
     k = 2 + 2
```
```python
i = n
while i > 0:
   k = 2 + 2
   i = i // 2
```
```python
for i in range(n):
   for j in range(n):
      for k in range(n):
         k = 2 + 2
```
```python
i = n
while i > 0:
   k = 2 + 2
   i = i // 2
```
```python
for i in range(n):
   k = 2 + 2
for j in range(n):
   k = 2 + 2
for k in range(n):
   k = 2 + 2
```
